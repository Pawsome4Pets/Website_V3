# Image assets

Drop your real photos here using these filenames and they will appear
automatically (the elegant placeholders disappear once a real file loads).

| Filename              | Used on                       | Suggested size      |
|-----------------------|-------------------------------|---------------------|
| logo.png              | Navbar logo (next to name)    | 200 × 200 (square)  |
| hero.jpg              | Home hero                     | 1200 × 1500 (4:5)   |
| about-1.jpg           | About — A Home, Not a Kennel  | 1000 × 800 (5:4)    |
| about-2.jpg           | About — Small & Toy-Breed Specialists | 1000 × 800  |
| about-3.jpg           | About — Care Without Compromise | 1000 × 800        |
| team-anzelle.jpg      | Team — Anzelle                | 800 × 1000 (4:5)    |
| team-anton.jpg        | Team — Anton                  | 800 × 1000 (4:5)    |
| slide-1.jpg .. slide-10.jpg | Home slideshow (10 photos) | 1600 × 1000 (16:10) |
| og-cover.jpg          | Social share image            | 1200 × 630          |

**Tip:** compress images (e.g. via squoosh.app) to under ~200 KB each for
fast mobile loading. Use WebP where possible. Slideshow photos look best
when shot in landscape (16:10 or similar).

To change any filename, edit `src/data/content.js`.
